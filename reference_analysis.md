# \[프로젝트 제안서\] AI 기반 당뇨병 예측 및 건강 상담 서비스

## 1\. 프로젝트 개요

본 프로젝트는 **Pima Indians Diabetes Dataset**을 활용하여 사용자의 당뇨병 발생 위험도를 예측하고, AI 챗봇을 통해 개인별 건강 관리 가이드를 제공하는 웹 서비스를 구축하는 것입니다. 기존의 단순 예측 시스템을 넘어, 사용자가 자신의 상태에 대해 실시간으로 질문하고 답변을 받을 수 있는 인터랙티브한 환경을 제공합니다.

## 2\. 배경 및 필요성

* **만성질환 관리의 중요성:** 당뇨병은 조기 발견과 지속적인 관리가 필수적인 만성질환으로, 일상적인 자가 점검 도구가 필요합니다.  
* **의료 데이터 활용의 가치:** 공개된 의료 데이터셋을 활용해 인공지능 모델의 실효성을 검증하고, 이를 실제 서비스화하는 경험은 기술적/사회적으로 가치가 높습니다.  
* **사용자 경험의 한계 개선:** 기존의 예측 서비스들은 결과 수치만 제공할 뿐, "결과가 왜 이렇게 나왔는지", "어떤 음식을 먹어야 하는지" 등의 후속 질문에 답하지 못하는 한계가 있습니다.

## 3\. 프로젝트 목표

* **정교한 모델링:** SVM(Support Vector Machine)과 데이터 전처리를 통해 높은 정확도의 당뇨 예측 모델을 구현합니다.  
* **대화형 인터페이스 구축:** LLM(거대언어모델) API를 연동하여 건강 관련 상담이 가능한 챗봇을 구현합니다.  
* **풀스택 개발 경험:** 데이터 분석부터 모델 학습, Flask/FastAPI 기반의 백엔드, 웹 프론트엔드까지 전체 파이프라인을 경험합니다.

## 4\. 기존 프로젝트 대비 개선점 (차별화 전략)

기존의 참고 프로젝트(`Diabetes-Prediction`)가 가진 단순 예측 구조를 다음과 같이 개선합니다.

| 구분 | 기존 프로젝트 (Aditya-Mankar) | 본 프로젝트 (개선안) |
| :---- | :---- | :---- |
| **핵심 기능** | 당뇨 여부 예측 (0/1) | **예측 \+ 건강 상담 AI 챗봇** |
| **데이터 활용** | 단순 수치 입력 및 결과 출력 | 예측 결과를 챗봇이 분석하여 **맞춤형 피드백** 제공 |
| **사용자 UI** | 기본적인 결과 페이지 | 대화형 채팅창을 통한 **양방향 소통** |
| **전처리 고도화** | 0값(결측치) 평균값 대체 | 이상치 제거 및 **특성 공학(Feature Engineering)** 추가 |

## 5\. 데이터셋 구성

* **출처:** Pima Indians Diabetes Dataset  
* **주요 변수:** 임신 횟수, 혈당, 혈압, 피부 두께, 인슐린 수치, BMI, 유전적 위험도, 나이 등 8개 항목  
* **전처리 방식:** 논리적 오류(0값)를 중앙값으로 대체하고, SVM 성능 극대화를 위해 **MinMaxScaler**를 적용합니다.

## 6\. 주요 기능

* **당뇨 위험도 예측:** 건강 데이터를 입력하면 모델이 위험도를 분석하여 시각화된 리포트를 제공합니다.  
* **AI 건강 어드바이저(챗봇):** 예측 결과와 연동되어 "당신의 혈당 수치가 높으니 식이요법이 필요합니다"와 같은 조언을 제공합니다.  
* **실시간 대화:** 사용자의 건강 관련 일반 질문(예: "당뇨에 좋은 운동은?")에 대해 실시간으로 답변합니다.

## 7\. 기대 효과

* **사용자 측면:** 병원 방문 전 간단한 자가 진단을 수행하고, 챗봇을 통해 올바른 건강 정보를 즉각적으로 습득할 수 있습니다.  
* **개발자 측면:** 머신러닝 모델의 서빙(Serving) 기법과 API 기반의 챗봇 연동 기술을 동시에 습득할 수 있습니다.  
* **확장성:** 향후 식단 사진 분석이나 운동량 트래킹 기능을 추가하여 종합 헬스케어 플랫폼으로 발전 가능합니다.
